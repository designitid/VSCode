Apa itu machine learning?

Machine learning adalah bidang yang memberikan kemampuan pada komputerr untuk belajar
tanpa di program secara langsung.

Pada pemograman biasa, orang harus memberikan input pada program dan program akan memproses input
tersebut untuk menghasilkan sebuah output.

Sementara pada machine learning, input dan output akan di proses sehingga menghasilkan input baru
yang akan menghasilkan output baru sampai seterusnya.

Bagaimana Machine Learning bekerja?

Machine learning bekerja dengan cara mengamati data yang sangat banyak untuk digunakan kembali.

Contoh Machine Learning di Sekitar kita
Computer Vision, Engine Rekomendasi, Penerjemah teks, Pembaca pola waktu, Pembaca anomali,.
